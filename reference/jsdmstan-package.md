# The 'jsdmstan' package.

A package for fitting joint species distribution models (jSDMs) in Stan.

This package can be used to both simulate and fit data according to a
multivariate generalised linear mixed model (MGLMM) or a generalised
linear latent variable model (GLLVM). All models are fit using the rstan
package. Summary functions are provided, as are interfaces to the
bayesplot plotting functions
