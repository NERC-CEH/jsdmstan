# Generate Stan code for a smoother from mgcv

This function creates Stan code for an `mgcv` smoother. It is (roughly)
based on [mgcv::jagam](https://rdrr.io/pkg/mgcv/man/jagam.html).

## Usage

``` r
make_smooth(sterms, data)
```

## Arguments

- sterms:

  a smooth term as generated by
  \[[mgcv::interpret.gam](https://rdrr.io/pkg/mgcv/man/interpret.gam.html)\]

- data:

  a `data.frame` that contains all the variables in `sterms`

## Value

a `list` of data to give to Stan
